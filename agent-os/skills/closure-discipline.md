# Skill: Closure Discipline

Use before a frame can close.

Required artifacts:
- evaluation.log
- review.log
- status
- bottleneck-feedback updates
- pushed commits

Required commands:
- focused tests
- full tests
- mypy
- docstrings
- wheel or frame-specific build gate
- any frame-specific gate

No closure if:
- review.log is missing
- any predicate is implied instead of verified
- uncommitted work exists in frame scope
- active bottlenecks lack prevention action
