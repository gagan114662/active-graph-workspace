## P17 — pentagon_watchdog_error investigation (DONE)
Read `~/.pentagon/trigger-bridge.err.log` (2.5 MB, 102 watchdog hits). Categorized:
- ~76 are **failed GET queries** (`/rest/v1/agent_triggers`, `/rest/v1/agents`) — the watchdog's
  stuck-trigger *detection poll* failing.
- 7 are **`fetch failed` / ECONNRESET** — transient network drops.

**Verdict:** the "watchdog errors" are NOT the recovery mechanism (osascript/kill/relaunch) failing —
they are **transient poll-query failures** (Supabase blips, JWT expiry, ECONNRESET) during the
stuck-trigger check. The watchdog retries on the next tick; nothing is actually broken. The earlier
"meta-defect: the recovery has its own failure mode" concern is **mostly benign network noise**, not a
broken watchdog.

**Remediation (low severity):** in the watchdog poll, treat GET failures as transient (skip the tick,
don't log at error level) and JWT-refresh on 401 — so the err log stops accumulating noise. Not urgent.

## P6 — grow judge ground-truth (skeptic review done; 0 promotable from current history)
Acted as skeptic-proxy on the harvested candidates. **All were synthetic test scaffolding** — diffs
that only append a comment like `# canonical-flywheel-test-marker` / `# flywheel-test-<ts>` to
`activegraph/__init__.py`, or a degenerate "not a valid diff at all". **0 are legitimate Rowan-FAIL
ground truth.** Promoting any would pollute the eval set with exactly the synthetic noise the
discipline forbids.

**Mechanism hardened:** `eval-harvest-from-failures.mjs::isSyntheticOrDegenerateDiff` now rejects
(a) non-diffs, (b) flywheel test markers, (c) no-op diffs whose only additions are blank/comment
lines. Re-harvest now yields 0 (clean) instead of 7 junk candidates.

**Honest conclusion:** the judge eval set **cannot grow from current gauntlet history** — it's almost
entirely synthetic test runs, not real agent work. Real growth is gated on **real, non-synthetic
flywheel cycles**, which need **RLS unblocked** (so real reviewer verdicts happen) + real dispatches.
This is the same keystone (Pentagon RLS / Option 0) blocking P2a. Until then, P6 grows only via the
operator hand-grading real examples (`scripts/grade-judge-example.mjs`).
